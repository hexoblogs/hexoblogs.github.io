Tout les versions de PUBG MOBILE

Created2021-03-25|Updated2021-03-26
|Post View:

En ce qui concerne les jeux mobiles, le genre battle-royale est incomplet sans
PUBG Mobile. Des matchs passionnants et des mises à jour régulières en font un
favori parmi des milliers de joueurs mobiles.

PUBG Mobile propose différentes versions dans différentes régions du monde. Mis
à part quelques changements mineurs, toutes les mécaniques de jeu et le gameplay
des versions sont les mêmes.

Liste des versions de PUBG Mobile disponibles pour différentes régions

Voici les versions de PUBG Mobile destinées à différentes régions du monde:

  1. PUBG Mobile (version mondiale)

Image via BlueStacks

Il s’agit de la version principale de PUBG Mobile, disponible dans la majorité
des pays. Lorsqu’il s’agit de grands tournois esports, cette version est
choisie.

2. PUBG Mobile KRJP

Image via Google Play Store

En ce qui concerne les versions régionales, celle-ci est la plus populaire. Le
jeu a sa propre devise en jeu connue sous le nom de Donkatsu Medal, qui peut
être utilisée pour acheter des skins et des accessoires.

3. PUBG Mobile VN

Image via Google Play Store

Il s’agit de la version vietnamienne du jeu, publiée par VNG Game Publishing.
L’interface utilisateur de la langue est passée au vietnamien au profit des
joueurs vietnamiens.

4. PUBG Mobile Taiwan

Image via pubg-mobile-Taiwan.agrariasundertaker.online

Cette version de PUBG Mobile a été publiée par HotCool Games, en gardant à
l’esprit les besoins des joueurs taïwanais. Le jeu peut être joué sur les
appareils Android et iOS à Taiwan.

5. Jeu pour la paix

Image via Firstpost

Ceci est la version chinoise du jeu. Game for Peace a été publié en 2019 par
Tencent Games, en gardant à l’esprit les restrictions du gouvernement chinois.
